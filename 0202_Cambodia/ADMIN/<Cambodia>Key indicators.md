# Cambodia Drug Report:Key Statistical Indicator (2012 - 2019)

This repository contains a summary of key indicators extracted from the annual reports published by the National Authority for Combating Drugs (NACD) of Cambodia.

As the reports are prepared in the local Cambodian language, the data has been included in this repository
